## 90-second version

Azazel-Edge is a deterministic edge decision appliance for constrained SOC/NOC
operations. It is designed for small, temporary, and rapidly deployed networks
where operators have limited infrastructure, limited personnel, unstable
connectivity, and very little time to decide.

The system separates NOC reliability context from SOC threat context, then uses
a bounded Action Arbiter to select one explicit response from `observe`,
`notify`, `throttle`, `redirect`, or `isolate`. It records the selected
evidence, the chosen action, the rejected alternatives, and the operator-facing
explanation in a local audit trail.

For the booth demo, deterministic replay is used to make the explanation and
audit path stable and inspectable in a short public session. Replay is a
presentation technique only; it does not replace the normal live Tactical
first-pass path used in operation.
