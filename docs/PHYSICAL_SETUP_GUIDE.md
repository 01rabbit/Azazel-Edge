# PHYSICAL SETUP GUIDE

## Purpose
Hardware assembly and cable routing reference.

Supplementary JA: 配線、ラベル、設置前確認をまとめた物理設置ガイド。

## Cable Routing (ASCII)
`[Power] -> [DC input]`
`[WAN/LAN] -> [eth uplink]`
`[Display] -> [EPD connector]`

## Port Labels
- `PWR`: main power
- `ETH-UP`: upstream
- `ETH-LAN`: local segment
- `EPD`: display connector

## Pre-deployment Checklist
- Enclosure fixed
- Cables secured
- Boot test passed
- Dashboard reachable
