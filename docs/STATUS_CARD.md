# STATUS CARD

Last updated: 2026-05-13

Quick response card for wall posting in field deployments.

---

## Print Instructions

- Size: A5 landscape (recommended)
- Material: laminate or water-resistant sleeve
- Placement: near gateway unit and operator desk line-of-sight
- Font target: readable from 2-3 meters

---

## Color States (Bilingual)

### GREEN / 緑
- Baseline stable.
- Keep monitoring.
- Follow normal checklist.

### YELLOW / 黄
- Check freshness and queue.
- Confirm affected scope.
- Avoid uncontrolled restarts.

### RED / 赤
- Treat as active incident.
- Follow reviewed runbook.
- Escalate immediately.

---

## Escalation Contact Format

Use the same pattern as operator contact sheet:
- Shift Lead: `+81-XX-XXXX-XXXX`
- Network Support: `+81-XX-XXXX-XXXX`
- Escalation Coordinator: `+81-XX-XXXX-XXXX`

Write clear local numbers before deployment.

---

## Minimum Disclosure Block

Post this card wherever users can see it:
- Monitoring is active for safety and service continuity.
- Follow operator instructions during degraded or incident states.
- Do not attempt unauthorized equipment changes.
