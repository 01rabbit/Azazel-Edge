# Summary

This folder converts the BHUSA 2026 Arsenal public description into implementation-ready readiness work.

The work is split into small deliverables because the accepted session should not be destabilized by a late large refactor. Each child issue should make the booth story clearer, more repeatable, or easier to recover.

Core exhibition story:

1. Constrained local network.
2. Evidence ingestion.
3. Separate NOC/SOC deterministic evaluation.
4. Bounded action arbitration.
5. Structured explanation with rejected alternatives.
6. Read-only audit review.
7. Replay for stable booth demonstration, not as a replacement for live Tactical operation.
