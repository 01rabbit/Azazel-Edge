# Booth output fields

The Vegas booth view should prioritize these fields:

- Current posture / mode
- Evidence summary
- Evidence IDs
- NOC evaluator result
- SOC evaluator result
- Selected bounded action
- Rejected actions
- Why chosen
- Why not others
- Policy profile
- Release condition
- AI Assist status
- Trace ID
- Audit chain verification status

These fields should be available either through Web UI, compact CLI, or JSON fallback.
