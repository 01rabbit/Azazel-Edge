# Non-goals before BHUSA 2026 Vegas

Parent roadmap: #283

These items are intentionally excluded from the Vegas readiness sub-issues:

- Full CTI platformization
- Full knowledge graph implementation
- Autonomous AI agent control
- Large internal refactor that risks demo instability
- Rebranding away from the accepted BHUSA 2026 title
- Cloud SOC replacement claims
- Production SIEM replacement claims
- New optional integrations that are not required for the accepted Arsenal story

The accepted story is deterministic edge decision support for constrained SOC/NOC operations. Anything that makes that story harder to demonstrate should be deferred.
