# MAINTENANCE CHECKLIST

## Purpose
Scheduled maintenance procedure checklist.

## Japanese Summary
年次・四半期・出動前の保守チェック項目です。

## Annual
- Full boot test
- Demo scenario run
- SD selftest
- Battery discharge verification

## Quarterly
- Check service status
- Check log rotation
- Review Suricata ruleset currency

## Per Deployment
- Physical inspection
- Cable verification
- Connectivity test
- Display verification
